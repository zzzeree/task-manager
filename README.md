# 📋 Менеджер задач — версия для интернета (24/7)

Эта версия работает **постоянно в интернете**, открывается **с любого устройства и браузера**
по одной ссылке, а задачи **общие на всех устройствах**.

- **Хостинг:** [Vercel](https://vercel.com) — бесплатно, без карты.
- **База данных:** [Upstash Redis](https://upstash.com) — бесплатно, без карты. Хранит все
  задачи как один JSON под ключом `tasks` (та же модель, что и в локальной версии).

Ничего устанавливать на компьютер не нужно — всё делается в браузере.

---

## Шаг 1. База данных Upstash (≈3 минуты)

1. Зайдите на **https://upstash.com** → **Sign Up** (можно войти через Google/GitHub).
2. Нажмите **Create Database** → выберите тип **Redis**.
3. Имя — любое (например `tasks`), регион — поближе (например Frankfurt/Europe). Создайте.
4. На странице базы найдите блок **REST API**. Скопируйте и сохраните два значения:
   - `UPSTASH_REDIS_REST_URL` — адрес вида `https://xxxx.upstash.io`
   - `UPSTASH_REDIS_REST_TOKEN` — длинный токен
   (Удобно: кнопка **.env** покажет обе строки сразу.)

## Шаг 2. Загрузить код на GitHub (≈3 минуты)

> Это нужно, чтобы Vercel взял проект. Всё — через сайт, без программ.

1. Зайдите на **https://github.com** → зарегистрируйтесь/войдите.
2. Нажмите **New repository** → имя `task-manager` → **Create repository**.
3. На странице репозитория: **uploading an existing file** (ссылка в середине).
4. Перетащите в окно **все файлы из папки** `task-manager-web`:
   `index.html`, `styles.css`, `app.js`, `package.json`, `.gitignore`
   и **папку** `api` (с файлом `tasks.js`).
5. Нажмите **Commit changes**.

## Шаг 3. Деплой на Vercel (≈3 минуты)

1. Зайдите на **https://vercel.com** → **Sign Up** → **Continue with GitHub**.
2. **Add New… → Project** → найдите репозиторий `task-manager` → **Import**.
3. Раскройте **Environment Variables** и добавьте две переменные (из Шага 1):

   | Name | Value |
   |---|---|
   | `UPSTASH_REDIS_REST_URL` | ваш адрес `https://....upstash.io` |
   | `UPSTASH_REDIS_REST_TOKEN` | ваш токен |

4. Нажмите **Deploy** и дождитесь окончания.
5. Готово! Vercel выдаст постоянную ссылку вида **`https://task-manager-xxxx.vercel.app`** —
   открывайте её на телефоне, планшете, другом компьютере. Задачи везде одни и те же.

---

## Перенести существующие задачи

Если уже добавляли задачи в локальной версии:
1. В локальной версии нажмите **💾 Сохранить JSON** — скачается `tasks.json`.
2. На сайте Vercel нажмите **⬆ Импорт JSON** и выберите этот файл.

## Как обновлять сайт потом

Изменили файлы → загрузите их снова в тот же репозиторий на GitHub (через **Add file → Upload files**).
Vercel автоматически пересоберёт и обновит сайт по той же ссылке.

## Возможные вопросы

- **«Internal Server Error» / пустой список** — проверьте, что обе переменные окружения в Vercel
  заданы верно (Project → Settings → Environment Variables), затем **Redeploy**.
- **Бесплатные лимиты** — Upstash Free: ~10 000 команд/день; Vercel Hobby: для личного
  некоммерческого использования. Для личного списка задач этого с запасом хватает.
- **Данные правда в JSON?** — да: весь список лежит в Upstash как одна JSON-строка под ключом
  `tasks`; формат совпадает с локальным `tasks.json`, файлы взаимозаменяемы.
